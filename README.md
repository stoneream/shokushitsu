# 職務質問 (shokushitsu)

これはあなたのプロダクティビティを測ります！！  

## これは何？

タスクの開始/終了を記録します。
1日の作業をプロジェクト/タスク単位で集計できます。

## 想定環境

- macOS
- Linux

## インストール

```bash
go install github.com/stoneream/shokushitsu/cmd/shoku@latest
```

## 使い方

### 起動方法

```bash
shoku
```

### ホーム画面

- `track`: 作業時間の計測を開始します
- `summary`: 日次サマリーを表示します
- `終了`: ツールを終了します

### タスク選択画面

`[+] 新規タスクを作成`  
プロジェクトとタスクを新規作成して開始します。

`最近使ったタスク一覧`  
過去に使ったタスクから選択して開始します。

### 新規作成する場合

1. プロジェクトを選択します（または新規作成します）。
2. タスク名を入力して Enter キーを押します。

### 作業中画面

プロジェクト名、タスク名、開始時刻、経過時間が表示されます。  
`e` で作業を終了します。  

25分ごとに通知と継続確認を行います。  
`c` を押すと継続します。  
15分入力がない場合は自動で継続として扱います。  

## 開発

- Go
- `just`（任意）

```bash
just build

# or

go build -o bin/shoku ./cmd/shoku
```

## その他

### 初回起動時に作成されるファイル

初回実行時に以下のファイルが作成されます。

- `~/.shoku/config.toml`
- `~/.shoku/shoku.db`

`SHOKU_HOME` を設定すると保存先を変更できます。
